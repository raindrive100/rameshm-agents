# rameshm-agents

This repository contains modules for implementing Autonomous Agents using OpenAI Agents SDK.

---

## Directory Hierarchy

### **pyproject.toml**
- Containts the various imports needed for "uv".

### **src/rameshm/agents/researcher/**
- guardrails.py: Module to implement Input Guardrail functionalityutilsctions_config.py: Module containing all the prompts for different agents.
- researcher_agent_with_openai.py: Reference implementation of Autonomous Agents using Input Guardrail, MCP Servers, and Handoff agent. 

---

### **src/rameshm/agents/utils/**
- basic_setup.py: Module to set-up environment, logging, and some paths to invoke uvx, nod etc..
